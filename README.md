# 🛒 E-Commerce Sales Performance Analysis using EDA

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on an E-Commerce Sales dataset obtained from Kaggle.  
The objective is to analyze sales performance, customer purchasing behavior, product trends, and revenue insights using Python-based data analysis techniques.

The analysis helps in understanding business growth patterns and identifying key areas for improving profitability.

---

## 🎯 Objectives

- Analyze overall sales performance
- Identify top-performing products and categories
- Study customer purchasing behavior
- Analyze regional sales distribution
- Detect seasonal sales trends
- Understand profit and revenue patterns

---

## 🗂 Dataset Information

- Source: Kaggle E-Commerce Dataset
- Format: CSV
- Records: (Add number of rows)
- Features: Order ID, Product Category, Sales, Profit, Quantity, Region, Order Date, Customer Segment, etc.

- Python

## 📊 EDA Operations Performed

- Data Cleaning & Handling Missing Values
- Data Type Conversion
- Duplicate Record Removal
- Summary Statistics Analysis
- GroupBy Aggregations
- Correlation Analysis
- Time Series Trend Analysis
- Data Visualization (Bar charts, Line charts, Heatmaps, Pie charts)

---

## 📈 Key Insights

- Identified top revenue-generating product categories
- Found highest profit margin segments
- Analyzed monthly & yearly sales growth
- Detected regions with maximum and minimum sales
- Observed customer purchasing patterns
